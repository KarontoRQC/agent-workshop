# EXP-20260617-agent-knowledge-graph

目标：做一个 66 个智能体知识图谱的可交互前端原型，用于智能体工作坊现场演示。

要验证的核心假设：局部焦点图谱比线性流程或全量星空更适合表达“从业务问题逐层展开到智能体组合”的体验。

输入：业务问题、行业、当前卡点、目标结果。

处理动作：以一个母节点为中心展开第二层节点，点击第二层节点后视觉焦点拉近，并将该节点变成新的母节点继续展开。

输出物：可运行的 React/Vite 原型，含模拟智能体推荐和可替换接口层。

成功标准：
- 首屏有全局规模感，但焦点路径清楚，不是满屏噪音。
- 点击节点时能明显感到焦点转移和逐层展开。
- 视觉接近浅色高级空间图谱：克制、立体、精致，而不是赛博大屏。
- 智能体跳转逻辑与前端图谱逻辑解耦。

失败信号：
- 变成线性流程图。
- 一开始展示过多节点导致不可读。
- 控制面板或标签把图谱主体压垮。

下一轮迭代：接入真实 66 个智能体清单和语义匹配接口。

可能归属子系统：智能体工作坊演示系统，当前仍在创意中心实验区。
